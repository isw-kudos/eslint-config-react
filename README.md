# eslint-config-react
huddo eslint config for react

.eslintrc.yaml:

```
  extends:
    - @huddo/eslint-config-react
```

## Getting Started

1. Install dependencies

    ```
    npm i
    ```

1. Login to NPM

    ```
    npm adduser
    ```

## Deploy to NPM

Update the package.json version number
Build `index.json` and publish to npm

```
npm run prepare
npm publish
```
